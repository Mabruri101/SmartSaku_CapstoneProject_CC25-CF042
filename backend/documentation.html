🛠️ Catatan Memulai Backend:
Untuk menjalankan backend API ini, lakukan langkah berikut di direktori proyek: 
    npm install
    npm start
Pastikan Anda sudah memiliki Node.js dan koneksi ke database (jika dibutuhkan).

📄 Dokumentasi API

Base URL: http://localhost:3000/api/user

1. 🔐 Login User

Endpoint: POST /login

Deskripsi: Autentikasi user berdasarkan email dan password.

Request Body:

{
  "email": "admin@test.com",
  "password": "admin123"
}

Response Body:

{
  "message": "Login berhasil",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "6840e85d41891a4e96746345",
    "name": "hacker",
    "email": "admin@test.com"
  }
}

Keterangan:

  • token: Token JWT untuk otorisasi selanjutnya
  • user: Data pengguna yang berhasil login

2. 📝 Register User

Endpoint: POST /register

Deskripsi: Mendaftarkan user baru ke sistem.

Request Body:

{
  "name": "hacker",
  "email": "admin@test.com",
  "password": "admin123"
}

Response Body:

{
  "message": "Registrasi berhasil",
  "user": {
    "id": "6843649d4669109495ce23f6",
    "name": "hacker",
    "email": "admin@test.com"
  }
}

Keterangan:

  • user: Data user yang berhasil didaftarkan
